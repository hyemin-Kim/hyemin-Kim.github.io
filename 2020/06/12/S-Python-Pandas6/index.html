4. (비)결측값 추출

4-1. 해당 column만 추출

결측값: df_name [ ‘col_name’] [ df_name [ ‘col_name’ ] .isna() / isnull() ]
비결측값: df_name [ ‘col_name’ ] [df_name [ ‘col_name’ ] .notna() / notnull()]
