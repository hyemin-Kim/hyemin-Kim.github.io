2. Fancy 인덱싱

fancy인덱싱은 범위가 아닌 특정 index의 집합의 값을 선택하여 추출하고 싶을 때 활용한다


2-1. 1차원 array

방법 1: 추출하고 싶은 index의 집합을 **[꺾쇠 괄호로]**묶어서 추출

array([23,  7, 65])

방법 2: 추출하고 싶은 index의 집합을 변수에 지정한 후 추출

array([23,  7, 65])


2-2. 2차원 array

array([[1, 2, 3, 4],
       [5, 6, 7, 8]])


array([[ 2,  4],
       [ 6,  8],
       [10, 12]])
