2. pandas의 Series 와 DataFrame

1차원, 1개의 column은 Series라고 한다

2-1. Series

Series 생성:

  • pd.Series(“list”)

  • pd.Series(“list_name”)


(1) pd.Series(“list”)

0    1
1    2
2    3
3    4
dtype: int64

(2) pd.Series(“list_name”)

0    1
1    2
2    3
3    4
dtype: int64


0    1
1    2
2    3
3    4
dtype: int64


2-2. DataFrame

방법 1. list로 만들기

0 1 2
0 삼성 2000 스마트폰
1 현대 1000 자동차
2 네이버 500 포털

<활용을 하기 위해 DataFrame을 변수에 지정하기>

0 1 2
0 삼성 2000 스마트폰
1 현대 1000 자동차
2 네이버 500 포털

<제목컬럼 만들기> – “dfname.column = [ ]”

기업명 매출액 업종
0 삼성 2000 스마트폰
1 현대 1000 자동차
2 네이버 500 포털

  • 주의: column명의 개수는 반드시 DataFrame의 column수와 동일해야 함

방법 2. dict로 만들기

2-3. index를 특정column으로 지정하기

“dfname.index = [ ]” 명령을 사용한다
