1. 함수란 무엇일까?

반복적으로 사용되는 부문을 묶어서, 재사용 가능하도록 만들어 주는 것

함수에는 **들어가는 놈 (input)**이 있고, **나오는 놈 (output 혹은 return)**이 있다. 전해진 로직(규칙)에 따라, input -> output으로 효율적으로 바꿔주는 역할을 한다
