4. 데이터의 응용

4-1. 사칙 연산자

연산자 의미 예
+ 더하기 2 + 1 -> 3
- 빼기 1 - 2 -> -1
* 곱하기 1 * 2 -> 2
/ 나누기 1 / 2 -> 0.5
// 몫 5 // 2 -> 2
% 나머지 5 % 2 -> 1
** 멱 2**3 -> 8
